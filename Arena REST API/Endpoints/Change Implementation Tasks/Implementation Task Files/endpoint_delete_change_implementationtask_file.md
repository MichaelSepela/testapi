# DELETE Change Implementation Task File
/changes/&lt;GUID&gt;/implementationtasks/&lt;GUID&gt;/files

Returns the implementation tasks for a specific change category. Note that Implementation Task Management and Object Level Override must be enabled for the category. If it is not enabled, the endpoints shall return an error indicating it is not enabled. Not supported for Canceled, Completed or Uncategorized changes.
